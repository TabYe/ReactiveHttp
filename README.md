### ReactiveHttp


#### 一个基于 Kotlin + Coroutines + Jetpack 封装的网络请求框架，详细的使用入门及原理讲解看这里：[wiki](https://github.com/leavesC/ReactiveHttp/wiki)

#### Download

```groovy
  allprojects {
    repositories {
      maven { url 'https://jitpack.io' }
    }
  }
	
  dependencies {
    implementation 'com.github.leavesC:ReactiveHttp:0.0.13'
  }
```
